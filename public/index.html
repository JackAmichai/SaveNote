The Future of SaveNote

This early browser MVP showcases the ideal native user experience. Our roadmap involves upgrading to a full, robust backend architecture:

📱 Native WhatsApp Sync

Transitioning from a browser extension to a dedicated, self-hosted Node application to enable seamless syncing directly across the native mobile WhatsApp app.

🧠 Deep Local LLM Integration

Replacing the keyword-based MVP with advanced NLP by securely bridging the app to a local Ollama API. The bot will understand full context without ever transmitting data to external clouds.
